Blogging With Octopress on GitHub

Jan 9th, 2013

This post is just a reminder to myself, how to publish new posts to my blog.

  1. rake new_post["Post Title"]
  2. edit post
  3. rake generate
  4. rake deploy
  5. git add .
  6. git commit -m 'some comments'
  7. git push origin source
